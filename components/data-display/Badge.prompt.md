Compact status / count label. Soft fill by default; `solid` for high emphasis.

```jsx
<Badge tone="success" dot>Live</Badge>
<Badge tone="brand" solid>3</Badge>
<Badge tone="ai">AI</Badge>
```

Tones: `neutral|brand|success|warning|danger|info|ai`. Props: `solid`, `pill`, `dot` (leading status dot), `size`, `icon`. Use `Tag` instead when the label is removable/filterable.
